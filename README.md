# Front sensor array

## Description
This code runs bare metal on Raspi Pi Pico.
<p>It collects data via I2C from an accelerometer / magnetometer</p>
<p>It also measures 2 suspension sensors</p>
<p>Parses data and sends it via CAN BUS</p>
